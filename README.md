# menu bot

The Menu Bot retrieves menus from Unicafe _(a student discounted lunch program in Finland)_ and shares them on Discord every Monday morning.

You can join the server here: [https://discord.gg/QtGdEVRKNc](https://discord.gg/QtGdEVRKNc)

![Demo screenshot](https://github.com/antoKeinanen/menu-bot/blob/main/img/demo.png?raw=true)

## usage

1. Ensure [bun](https://bun.sh/) is installed.
1. Install dependencies by running `bun install`.
2. Set up a Discord server with the necessary channels.
3. Add menus to the `config.ts` file.
4. Create a `.env` file containing all the webhook URLs.
5. Execute `bun run start` to run the bot.
